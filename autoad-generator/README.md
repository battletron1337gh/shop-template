# 🚗 AutoAd Generator

Een web-gebaseerde tool voor het genereren van professionele occasion advertenties voor Marktplaats, Autoscout24 en andere verkoopplatformen.

## ✨ Features

- 📱 **Mobiel-vriendelijk design** - Werkt perfect op smartphone, tablet en desktop
- 📝 **Slim formulier** - Vul alle auto-details in met een overzichtelijk formulier
- 🎯 **Drie advertentiestijlen** - Marktplaats, Autoscout24 en Algemeen
- 💾 **Lokale opslag** - Sla advertenties op in je browser
- 📋 **Kopieerfunctie** - Kopieer direct naar je klembord
- 🎨 **Modern design** - Frisse, professionele uitstraling

## 🚀 Starten

```bash
node server.js
```

De server start op `http://0.0.0.0:8080`
